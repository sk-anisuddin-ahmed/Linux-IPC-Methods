# Linux IPC Methods

Minimal, working examples of Linux inter‑process communication (IPC) techniques. 

---

## 📂 Methods Covered

- **D‑Bus**
  - Example service and client demonstrating message bus communication.
  - Shows how to register a service, handle method calls, and invoke methods using `busctl`.

- **Sockets**
  - Simple TCP/UDP socket programs for process‑to‑process communication.
  - Includes both server and client examples.
